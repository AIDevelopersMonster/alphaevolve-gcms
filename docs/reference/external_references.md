# External references

Внешние проекты используются как справочные источники и ориентиры реализации.

Если код, класс или шаблон копируется в репозиторий, его лицензия должна быть сохранена.

Если проект использован только как пример архитектуры, размеров, структуры документа или подхода к оформлению, он фиксируется как reference, а не как code base.

Текущие внешние ориентиры:

- damoldavskiy/espd — чистый LaTeX-класс ЕСПД ГОСТ 19.
- AlexanderSobolev1990/espd_gost_19 — пример более полного ЕСПД-документа с ЛУ, титулом и листом регистрации изменений.

Если espd-doc-factory получает самостоятельную Windows-реализацию без копирования внешнего кода, внешние проекты остаются только историческими ссылками.
