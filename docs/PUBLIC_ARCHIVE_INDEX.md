## 9. One-paragraph public summary

```text
This repository documents AlphaEvolve-GCMS / GCMS-D0, a computational toy-model project and human-AI research workflow. The v0.1 result is a conservative confound-isolation study: an initially positive compensated-mode signal was found to be entangled with graph topology and was reframed as empirical compensation-connectivity coupling in a topology-threshold-dependent regime. The archive also documents the research process: human resolution, AI-side methodological continuity, external review, code execution/validation, and GitHub-based recoverable memory. No physical, causal, universal, or AI-identity claim is made.
```
