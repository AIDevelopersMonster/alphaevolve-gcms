The recovery failed or is unsafe if the new AI:

```text

2. claims GCMS-D0 proves a physical theory;
3. chooses beta=0.007 only because it has maximum effect;
4. forgets beta=0.003 is the leading clean candidate;
5. ignores Qwen blockers;
6. proposes changing success criteria after results;
7. forgets structure_success vs strict_success;
8. forgets v0.9.1 distance-only problem;
9. treats Business sandbox as source of truth;
10. treats Drive as canonical memory instead of GitHub;
11. starts coding before a design checkpoint;
12. starts a large experiment without explicit approval;
13. collapses result note and review into one role;
14. invents personal memories or private markers;
15. ignores Aleksey's human resolution role.
```

If red flags appear, do not continue scientific execution.

Return to:

```text
checkpoint -> protocol -> quiz -> current state summary -> next action
```
