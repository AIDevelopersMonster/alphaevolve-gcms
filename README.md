# AlphaEvolve-GCMS / GCMS-D0

AlphaEvolve-GCMS / GCMS-D0 is a computational toy-model research project for studying globally compensated multi-index systems and emergent local graph structure.

This repository is also a documented human-AI research workflow archive: it records how hypotheses, experiments, reviews, reframes, and continuity protocols were preserved through GitHub documents.

## Current v0.1 conclusion

The main v0.1 result is conservative.

An initial compensated-mode signal in Variant 2 at `beta=0.003` showed higher `structure_success` than the uncompensated reference, but subsequent controls showed that the signal was entangled with graph topology.

The current interpretation is:

```text
In this toy model, compensated Variant 2 at beta=0.003 is best interpreted as a connectivity-threshold-dependent study regime. The apparent compensated advantage is empirically coupled to native topological capacity, especially non_bridge_edge_count and cycle_rank. This supports a confound-isolation / topology-threshold framing, not a density-independent proof.
```

## Core idea

Global compensation:

```text
sum_B I_i ~= 0
```

Local sector:

```text
sum_U I_i = K != 0
```

Working question:

```text
Can globally compensated multi-index systems generate or preserve long-lived local graph sectors with statistically non-random structure?
```

The v0.1 answer is not a confirmation of a clean compensation mechanism. It is a documented confound-isolation result showing that apparent compensated advantage must be interpreted through native graph topology.
